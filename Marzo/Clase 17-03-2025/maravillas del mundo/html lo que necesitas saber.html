¿Qué es HTML?

El codigo HTML(sigla del imglés Hypertext Marking Language)se puede traducir libremente como lenguaje de Marcado de Hipertexto, lo que permite el uso de tags para estructurar y formatear páginas web. Puedes agregar elementos como:

  • Texto simple o listas de palabras
  • imágenes
  • Hiperlinks
  • Tablas
  • la estructura básica de una pagina tiene
  • doctype:informa que estás utilizando un lenguaje HTML
  • headinicio de todo, que contiene todo el contenido que se desarrollara
  • meta charset
