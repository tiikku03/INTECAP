Audio

La etiqueta -audio- permite incrustar archivos de audio en las páginas. Para usar esta etiqueta, especifique la ruta del archivo de audio en el atributo src. Por ejemplo, si desea incluir un archivo de audio de un CD en la página, deberá introducir cd_audio como valor de origen en el atributo src. También puede especificar "auto" en src para que el navegador seleccione automáticamente el archivo de origen adecuado.

Video

La etiqueta -video- permite incrustar vídeos en las páginas. Para usar esta etiqueta, especifique una URL o un ID (un identificador único) para el elemento de vídeo. La URL debe ser relativa. Por ejemplo, si desea incluir un vídeo de YouTube en la página, deberá introducir "youtube_id" como valor del atributo id. Si no se especifica ninguna URL, se utilizará Adobe Flash Player para cargar el vídeo de YouTube. (Para obtener más información sobre el uso de Adobe Flash Player con páginas HTML5, consulte este artículo).
